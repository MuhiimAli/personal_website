BERT Question-Answering System

Bert NLP Python

This project implements a fine-tuned BERT model for Question Answering (QA) that can extract precise answers from given context passages or determine if no answer exists. The system achieves competitive performance metrics (P=61.3%, R=47.9%, F1=53.8%) by implementing custom loss functions and sophisticated data preprocessing techniques based on the "A BERT Baseline for the Natural Questions" paper.

Reddit Depression Detection

NLP Machine Learning

Built a machine learning system that helps detect signs of depression in Reddit posts. The project analyzes text data using two different methods - LDA topic modeling and RoBERTa embeddings - to train a set of random forest classifiers. Each classifier is trained to recognize specific symptoms by comparing mental health-related posts against regular social media content.

Spring 2024

Craigslist Item Price Predictor

Data Engineering & Analysis Machine Learning

Created a machine learning system to predict second-hand item prices on Craigslist using multi-modal data analysis. The project combines computer vision and NLP techniques by processing both item images and text descriptions. Using a gradient boosting regressor, the system analyzes features extracted from VGG16 for images and spaCy embeddings for text, along with categorical data like location and item category to predict market prices.

Forge

Built an interactive visualizer to explore the Gale-Shapley matching algorithm, used in systems like college admissions and job matching. Created using Forge modeling language, the project demonstrates how the algorithm finds optimal matches between two groups while examining interesting behaviors like strategic preference ranking.
